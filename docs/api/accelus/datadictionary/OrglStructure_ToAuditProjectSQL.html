SQL Example for: Audit Projects, Organizational Structure
SELECT OrglStructure.DocumentID,AuditProject.DocumentID
FROM
OrglStructure
JOIN OrglStructure_ToAuditProject ON OrglStructure.DocumentID = OrglStructure_ToAuditProject.DocumentID
JOIN AuditProject ON AuditProject.DocumentID = OrglStructure_ToAuditProject.ToAuditProject
