SQL Example for: Key Risk Indicator Details, Key Risk Indicator Values
SELECT KRIInputValue.DocumentID,KRIInputDetails.DocumentID
FROM
KRIInputValue
JOIN KRIInputValue_ToKRIInputDetails ON KRIInputValue.DocumentID = KRIInputValue_ToKRIInputDetails.DocumentID
JOIN KRIInputDetails ON KRIInputDetails.DocumentID = KRIInputValue_ToKRIInputDetails.ToKRIInputDetails
