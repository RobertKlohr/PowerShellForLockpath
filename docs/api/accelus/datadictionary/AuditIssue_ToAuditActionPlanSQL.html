SQL Example for: Audit Action Plans, Audit Issue
SELECT AuditIssue.DocumentID,AuditActionPlan.DocumentID
FROM
AuditIssue
JOIN AuditIssue_ToAuditActionPlan ON AuditIssue.DocumentID = AuditIssue_ToAuditActionPlan.DocumentID
JOIN AuditActionPlan ON AuditActionPlan.DocumentID = AuditIssue_ToAuditActionPlan.ToAuditActionPlan
